Globale tags på hjemmesiden

            
                
Bruges til style af selve Body


                * {
                    padding: 0;
                    margin: 0;
                    box-sizing: border-box;
                }

                body{
                    background-color: #FDFEFF;
                  }

            
          
            
                
Alle button tag få cursor: pointer


                button {
                    cursor: pointer;
                }

            
          
            
                
Header på alle sider har samme grøn farve


                header {
                    background-color: #3DCD58;
                }

            
          
            
                
Alle a tags får den styling


                a {
                    color: white;
                    text-decoration: none;
                    font-family: arial Rounded MT;
                }

            
          
            
                
Alle h1 tags får den styling på alle sider


                h1 {
                    font-family: arial Rounded MT;
                    font-size: 170%;
                    color: white;
                }


            
          
            
                
Alle h2 tags får den styling på alle sider


                h2 {
                    font-family: arial Rounded MT;
                    font-size: 200%;
                    color: #1F1F1F;
                    margin-bottom: 7px;
                }


            
          
            
                
Alle h3 tags får den styling på alle sider


                h3 {
                    font-family: arial Rounded MT;
                    font-size: 140%;
                    color: #1F1F1F;
                }


            
          
            
                
Alle p tags får den styling på alle sider


                p {
                    font-family: arial;
                    font-size: 100%;
                    color: #1F1F1F;
                }
